# Bankist - Fake Bank Website

Welcome to Bankist, a fictional online banking platform where you can explore various banking features. 
This project was developed as part of Jonas Schmedtmann's JavaScript course on Udemy, with the JavaScript code written independently.

## Features

- **User Authentication:** Log in to different accounts using usernames and PINs.
- **Account Overview:** View different banking movements in various currencies.
- **Money Transfer:** Transfer money between different accounts securely.
- **Summary:** Get a summary of all incoming, outgoing, and interest transactions.
- **Sorting:** Sort your money movements for better organization.
- **Date Filtering:** Check transactions for today, yesterday, N days ago, or a specific date.
- **Inactivity Timer:** Automatically logs you out after a period of inactivity.

## How to Use

1. Clone the repository to your local machine.
2. Open the "index.html" file in your web browser to access the Bankist website.
3. Use the provided usernames and PINs to log in and explore the features.

## Project Structure

- **HTML:** The main structure of the web page.
- **CSS:** Styling to create an appealing user interface.
- **JavaScript:** Logic for user authentication, banking features, and UI updates.
